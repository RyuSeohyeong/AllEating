# AllEating
스프링 부트를 활용한 식품 쇼핑몰 웹페이지 프로젝트

# 개발기간
- 23.08.01 - 23.09.08

# 맴버구성
- 팀장: 최현진 - 로그인, 회원가입, ID찾기, PW찾기 ,상품등록&수정, 장바구니, 결제페이지, 관리자페이지
- 팀원1: 류서형 - 메인페이지, 마이페이지
- 팀원2: 조승기 - 메인페이지, 사업자페이지
- 팀원3: 공지사항

# 개발 환경
- java : 1.8
- IDE : STS4
- Framework : Springboot(2.7.13)
- Database : MySQL DB(8.0)
- ORM : Mybatis
